# 📒 Command-Line Contact Book Application

## 📌 Project Description
This is a simple Command-Line Contact Book Application developed using Python. It allows users to store and manage contact information in a text file.

## ✨ Features
- Add new contacts
- View all saved contacts
- Search contacts by name
- Store data permanently using `contacts.txt`
- Easy-to-use menu-driven interface

## 🛠️ Technologies Used
- Python 3
- File Handling
- Functions
- Conditional Statements
- Loops

## 📂 Project Structure
```
ContactBookProject/
│── contact_book.py
│── contacts.txt
│── README.md
```

## ▶️ How to Run
1. Open the project in VS Code.
2. Open the terminal.
3. Run:
```bash
python contact_book.py
```

## 📸 Sample Menu
```
1. Add Contact
2. View Contacts
3. Search Contact
4. Exit
```
